# README

`@umijs/max` 模板项目，更多功能参考 [Umi Max 简介](https://umijs.org/docs/max/introduce)

采用React 18、Ant Design Pro 6、TypeScript 5。

支持加载动态权限菜单，多方式轻松权限控制。

## 前端开发注意事项

​	Node：建议v16或以上

​	安装依赖请支行：pnpm i

​	正常启动请运行: pnpm run dev

​	Mock测试模式请运行: npm run start

​	发布打包请运行: npm run build
